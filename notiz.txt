Scroll Пропсы:

- scrollTop:
  • number:
  • "end":
    если значение "end" то для более точного обновления при изменении объектов скролла
    сравнивается ключ первого объекта (это для подгрузки элементов сверху)

- thumbElement:
  • boolean:
    при значении false отображает дефолтный скролл браузера
    если передать true то отобразится дефолтный элемент для прокрутки
  • React.ReactNode:
    в дефолтный элемент можно передать свой стилизованный компонент прокрутки

- onScrollValue?: Array<(scroll: number) => boolean>;
  это колбек для вызова при удобного вызова при определённых значениях скролла:
  onScrollValue={[
    (scroll) => scroll < 200 && console.log("scroll < 200"),
    (scroll) => scroll > 200 && console.log("scroll > 200"),
  ]}

- isScrolling?: (status: boolean) => void;
  колбек функция для передачи boolean состояния прокрутки

- infiniteScroll?:
  • reezeOnScroll?: boolean;
    значение для остоновки рендера элементов при прокрутке, элементы автоматически
    заменяются на null или fallback (если его передали)
    что бы загрузка прекращалась только для незагруженных элементов отслеживаются элементы
    с атрибутом "data-id" значение которого id скролла и id  объекта
